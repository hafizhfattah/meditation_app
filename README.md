### 📦 **Meditation App**

MedtApp Is A Meditation App With Basic Fiture.

### 🎁 **Fiture**
- Icon Logo
- App Name
- Splash Screen

### 📄 **Page Include**
- Home Page
- Meditation Page
- Calendar Page
- Easy State Management
- Animation Lottie
- Calendar

### 🚀 **Getting Started**
1. Clone this project || Download Zip from GitHub
2. Run `flutter pub get` In Terminal
3. Run the project In Web or Emulator
